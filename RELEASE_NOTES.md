# v2.0.0 Release Notes

## 🚀 Major Features

### 1. 과제 톹계 및 대시보드 (Assignment Tracker)
- **메인 페이지 통계**: LMS 메인에서 각 강의별 과제 현황(완료/미완료)을 한눈에 볼 수 있습니다.
- **강의별 대시보드**: 강의 페이지 상단에 남은 과제, 마감 임박 과제 등을 요약하여 보여줍니다.
- **스마트 상태 표시**: '제출완료', '마감임박', '지각' 등의 상태를 색상 칩으로 명확하게 표시합니다.

### 2. PDF 다운로드 (PDF Downloader)
- **자동 감지**: LMS 내 문서 뷰어 페이지 진입 시 자동으로 다운로드 버튼이 활성화됩니다.
- **이미지 → PDF 변환**: 이미지 조각으로 된 교재를 하나의 PDF 파일로 합쳐서 다운로드합니다.

## 🛠 Improvements
- **프로젝트 구조 개편**: 기능별(Feature-based)로 코드를 분리하여 유지보수성을 높였습니다.
- **아이콘 리뉴얼**: 새로운 해상도별 아이콘이 적용되었습니다.

## 📦 Installation
1. `Assets` 항목에서 `csmsplus-v2.0.0.zip` 파일을 다운로드합니다.
2. 압축을 해제합니다.
3. Chrome 주소창에 `chrome://extensions`를 입력합니다.
4. 우측 상단 '개발자 모드'를 켭니다.
5. '압축해제된 확장 프로그램을 로드합니다'를 클릭하고 압축 해제한 폴더를 선택합니다.
